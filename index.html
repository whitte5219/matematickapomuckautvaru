Pomůcky

Tato část slouží k pochopení rozdílu mezi pojmy obvod, obsah, povrch a objem.

Význam pojmů

Obvod (2D)

Obvod je délka čáry kolem rovinného útvaru.

Obsah (2D)

Obsah je plocha uvnitř rovinného útvaru.

Povrch (3D)

Povrch je součet ploch všech stěn tělesa.

Objem (3D)

Objem je prostor uvnitř tělesa.

Vzorce pro jednotlivé tvary

Čtverec
  • Obvod = 4 · a
  • Obsah = a²
Obdélník
  • Obvod = 2 · (a + b)
  • Obsah = a · b
Kruh
  • Obvod = 2 · π · r
  • Obsah = π · r²
Trojúhelník
  • Obsah = (a · vₐ) / 2
  • Obvod = a + b + c
Krychle
  • Povrch = 6 · a²
  • Objem = a³
Kvádr
  • Povrch = 2 · (a·b + a·c + b·c)
  • Objem = a · b · c
